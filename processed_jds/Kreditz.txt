RESPONSIBILITES

Some Of The Main Responsibilities You Will Cover

Design, build, and maintain cloud infrastructure using Infrastructure as Code (Terraform).
Develop and optimize CI/CD pipelines for automated testing and deployment.
Build and deploy services on AWS using ECS, Fargate, and Service Connect.
Implement and maintain monitoring, alerting, and observability solutions.
Collaborate with developers to improve deployment workflows and platform reliability.
Work with serverless technologies (Lambda) for event processing and automation.
Perform code reviews and contribute hands-on to infrastructure projects.
Leverage AI agents and tools to enhance development workflows and productivity.
Document infrastructure architecture, processes, and runbooks.

Success means engineers ship confidently, systems scale without drama, and incidents are rare, visible, and quickly resolved.
